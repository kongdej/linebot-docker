# linebot-docker
 
### Installation
- Prerequisite
    - Install Git
    - Create LINE BOT and LINE Login
    - Install ngrok

- Download
```
    git clone https://github.com/kongdej/linebot-docker
```

- Update .env
```
    HOST =  https://9086-1-46-203-66.ngrok-free.app
    PORT = 8000

    BOT_ACCESS_TOKEN = xxxxxxxxxxxxxxxxxxxxxxxxx
    BOT_SECRET = xxxxxxxxx
```   

- Run docker compose
```
   docker compose up
```
